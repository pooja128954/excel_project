# 📊 E-Commerce Sales & Delivery Insights Dashboard (Excel Project)
# 📌 Project Overview
This project is an interactive E-Commerce Sales & Delivery Insights Dashboard built entirely in Microsoft Excel.
The dashboard provides a clear and data-driven overview of:
	• Total Sales Performance
	• Average Order Value
	• Best Selling Categories
	• Regional Sales Distribution
	• Monthly Sales Trends
	• Top 5 Customers
	• Average Delivery Time by Region
The goal of this project is to demonstrate data analysis, dashboard design, and business insight generation using Excel.

# 🎯 Objectives
	• Analyze overall sales performance
	• Identify top-performing product categories
	• Track monthly revenue trends
	• Evaluate regional sales contribution
	• Monitor delivery performance
	• Identify high-value customers

# 🛠️ Tools & Techniques Used
	• Microsoft Excel
	• Pivot Tables
	• Pivot Charts
	• Slicers (Region & Product Category filters)
	• KPI Cards (using formulas)
	• Data Cleaning & Aggregation
	• Dashboard Formatting & Layout Design
